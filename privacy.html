2. 위치정보 처리

위치 정보는 브라우저의 Geolocation API를 통해 수집되며, 서버에 전송·저장되지 않습니다. 수집된 좌표는 카카오 장소 검색 API 호출 시 반경 탐색 파라미터로만 사용되며, 사용자 식별에 활용되지 않습니다.

4. 쿠키 및 로컬스토리지

서비스는 쿠키를 사용하지 않습니다. 브라우저 로컬스토리지에 즐겨찾기·개인정보 동의 여부만 저장하며, 브라우저 설정에서 언제든지 삭제할 수 있습니다.

5. 정보 보유 및 파기

서버에 저장되는 개인정보가 없으므로 별도의 파기 절차가 필요하지 않습니다. 기기 내 로컬스토리지 데이터는 브라우저 설정 → 사이트 데이터 삭제를 통해 직접 삭제할 수 있습니다.

6. 이용자의 권리

서비스는 개인정보를 서버에 보관하지 않으므로 별도의 열람·정정·삭제 요청 절차가 없습니다. 로컬스토리지 데이터 삭제는 브라우저에서 직접 처리하실 수 있습니다.
